================
FP8
================

FP8 is second generation the mobile counterpart of the :doc:`/AMD/AM5` socket and uses the :doc:`/Memory/DDR5` memory standard.
This socket's predecessor was :doc:`FP7`.

CPU Familes on this socket:
================================

* Ryzen 7000 (Zen 4)
* Ryzen 8000 (Zen 4)
* Ryzen AI (Zen 5)
