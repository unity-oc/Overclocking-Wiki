====================
FM1
====================

Introduction
====================

AMD's first APU socket, using the :doc:`/Memory/DDR3` memory standard.
It was succeeded by :doc:`FM2`.

Key Facts & Figures
====================

.. list-table::
   :widths: 50 75
   :header-rows: 0

   * - Microarchitecture:
     - :doc:`/AMD/Microarchs/Bulldozer`
   * - Process nodes:
     - GlobalFoundries 32nm or 28nm
   * - Core counts:
     - 2, 4, 6 or 8 (no SMT)
   * - Native FSB:
     - 200 MHz
   * - Chipsets:
     - AMD A55 and A75
