====================

   :widths: 50 75
   :header-rows: 0

   * - Microarchitecture:
     - :doc:`/AMD/Microarchs/Bulldozer`

     - 2, 4, 6 or 8 (no SMT)
   * - Native FSB:
     - 200 MHz
